# Integrated Food Delivery and Dineout Hospitality

A full-stack web application that combines food delivery and dine-out table booking into one seamless platform.

## Features
- 🍔 Food ordering and delivery
- 🍽️ Dine-out table reservations
- 🔐 User authentication
- 🛒 Cart management
- 📦 Order tracking
- 💳 Payment integration

## Tech Stack
- Frontend: React.js
- Backend: Node.js / Express
- Database: Firebase
- Styling: CSS / Tailwind

## Getting Started

```bash
npm install
npm start
```

## Project Structure
- `/frontend` - React frontend
- `/backend` - Node.js backend
- `/admin` - Admin dashboard
